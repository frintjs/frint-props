# Guide

## Installation

With [npm](https://www.npmjs.com/):

```
$ npm install --save frint-props
```

Via [unpkg](https://unpkg.com) CDN:

```html
<script src="https://unpkg.com/frint-props@latest/dist/frint-props.min.js"></script>
<script>
  // available as `window.FrintProps`
</script>
```
